# codecamp4

## How to run as 4-step

```bash
  ### Step 1/4: Clone this project
  git clone https://github.com/buildingwatsize/codecamp4.git

  ### Step 2/4: Install Package
  npm install

  ### Step 3/4: Change directory into code
  ### Example src/day1 ###
  cd src/day1

  ### Step 4/4: Run test
  npm run test
```
